# GarageDoor
Rudimentary frontend for PL Garage

# Set-up
Beside the EXE in releases (you may have to compile this yourself with Linux), make a file called config.txt
and format like this:
```
BaseUrl=https://your-plg-instance.cool
Port=8080
```

You will want to change these values.
